# OST Vendor Notes

배금도시 v2 OST는 PEPEANT/dis-site의 RECLAIM_2.5 BGM 원본 mp3를 사용한다.

## Source

- Repository: https://github.com/PEPEANT/dis-site
- Source folder: `games/RECLAIM_2.5/bgm`
- Imported folder: `assets/audio/ost/reclaim-2.5`

## Imported Files

- `reclaim-2.5/bgm_0.mp3`
- `reclaim-2.5/bgm_1.mp3`
- `reclaim-2.5/bgm_2.mp3`

`RECLAIM_1.0/bgm`의 `bgm_1.mp3`, `bgm_2.mp3`는 `RECLAIM_2.5`와 중복이라 별도 복사하지 않았다.
